# End-User Docs

This folder is for practical usage guides aimed at library and CLI users.
The implementation code is the source of truth for these pages.

Each surface folder contains:

- a `README.md` overview
- a grouped `reference.md` inventory for public functions and config families
- task-focused guides where a feature needs more examples

- [Python](./python/README.md)
